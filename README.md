# Senai-LIMA-1DN

<tr> -> cria cabeçalho da coluna (exemplo títulos)
<th> inclui dados no cabeçalho (exemplo "nome", "cpf" e "turno" fica em negrito)
<td> dados das linhas da tabela

A estrutura fica:
<table>
            <tr>
                <th></th>
            </tr>
            <tr>
                  <td></td>
            </tr>
</table>
